# MindWatch

AI-Based Social Media Mental Health Risk Detection and Emotional Trend Analysis

## Features
- Severe mental health risk detection using DeBERTa
- Fine-grained emotion analysis using GoEmotions
- LOW / MEDIUM / HIGH risk classification
- Explainable AI with SHAP
- Risk keyword highlighting
- Confidence scoring
- Interactive Flask web dashboard

## Tech Stack
Python, Flask, PyTorch, Hugging Face Transformers, SHAP, HTML, CSS

## Performance
Severe Risk Detector:
- Accuracy: 90.65%
- Macro F1: 75.76%

## Run
```bash
pip install -r requirements.txt
python app.py
